# Personal Fitness Tracker

A simple and interactive **Personal Fitness Tracker Web App** built using Python and Streamlit.
This app helps users track calories burned based on different exercises.

---

## Features

* Calculate calories burned
* Track different exercises
* Interactive UI using Streamlit
* Uses CSV datasets for exercise & calorie data
* Fast and lightweight web app

---

## Technologies Used

* Python 
* Streamlit
* Pandas
* NumPy

---

## Project Structure

```
project/
│
├── app.py
├── calories.csv
├── exercise.csv
├── requirements.txt
```

---

## How to Run Locally

1. Clone the repository:

```
git clone https://github.com/your-username/fitness-tracker.git
cd fitness-tracker
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
streamlit run app.py
```
